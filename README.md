## SQL – LibraryTX Project

This repository includes an SQL practice project called **LibraryTX**, where I created:

- `library_database.sql` – database schema and sample data  
- `library_queries.sql` – SELECT queries (filters, joins, aggregation)  
- `library_updates.sql` – UPDATE, DELETE, CASE, JOIN, LEFT JOIN practice  

The project covers: basic SQL queries, JOINs, filtering, aggregation, subqueries, and data updates.
