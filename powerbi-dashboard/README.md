## Overview
Interactive Power BI dashboard analyzing salaries across data roles.

## Key insights
- Median salary by experience level
- Salary differences by company size
- Global salary distribution (map)
- Salary trends over time
- Role-based filtering (Top 10 Job Titles)

## Features
- Dynamic slicers
- Interactive map  
  (bubble size = number of records, color = median salary)
- Tooltips with detailed values
- All visuals dynamically filtered by selected job title

## Tools
- Power BI
- DAX
- Data modeling

## How to use
Download the `.pbix` file and open it in Power BI Desktop.
