# Power BI – Salary Analysis Dashboard

![Dashboard preview](dashboard_preview.png)

## Overview
Interactive Power BI dashboard analyzing global salary data across data-related roles.
The dashboard allows users to explore salary differences by experience level, company size, job role, and geography.

## Key insights
- Median salary by experience level (Junior → Executive)
- Salary differences by company size (S / M / L)
- Global salary distribution by country (map)
- Salary trends over time
- Role-based filtering (Top 10 Job Titles)

## Features
- Dynamic slicers (job title, contract type)
- Interactive world map  
  *(bubble size = number of records, color = median salary)*
- Tooltips with detailed values
- All visuals dynamically filtered by selected job title

## Tools
- Power BI
- DAX
- Data modeling

## How to use
Download the `.pbix` file and open it in **Power BI Desktop**.

## Dataset
The dashboard is based on a publicly available dataset containing global salary data for data-related roles.
